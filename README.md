# 🛡️ Guardian Band – Smart Women's Safety Wearable System

## 📖 Overview

Guardian Band is an innovative wearable safety solution designed to enhance women's security during emergency situations. The system combines IoT, GPS tracking, voice recognition, and real-time communication technologies to provide immediate assistance when the user is in danger.

The wearable band allows users to trigger emergency alerts through a panic button or voice command. Once activated, the system shares the user's live location and sends SOS notifications to registered guardians via SMS and email.

---

## 🚨 Problem Statement

Women often face safety concerns while traveling alone or during emergencies. Existing solutions may not provide quick response mechanisms or real-time location tracking. There is a need for a smart, reliable, and easy-to-use wearable device that can instantly notify trusted contacts and share location information during critical situations.

---

## 🎯 Objectives

- Enhance women's safety using wearable technology.
- Provide one-touch emergency assistance.
- Share real-time GPS location with guardians.
- Support voice-based emergency activation.
- Deliver instant SMS and email alerts.
- Improve emergency response time.
